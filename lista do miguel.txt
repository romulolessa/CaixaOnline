Questão 16:A- O algoritmo representado é o algoritmo Bubble Sort. Esse algoritmo compara  os elementos de uma lista de tamanho n dois a dois. Isso ocorre da seguinte maneira: em cada iteração, o elemento de uma posição i é comparado com o elemento da posição i+1. Caso na comparação seja verificado que os elementos estão fora da ordem estabelecida, as posições desses elementos são trocadas. 

Questão 17: B- Insertion Sort, os elementos são ordenados da esquerda para direita. Nesse algoritmo, elege-se uma chave (a primeira chave é o segundo elemento da lista) para realização das comparações. Em cada iteração, a chave vai avançando para o terceiro, quarto, quinto até o último elemento, de modo que quando se realiza a operação sobre o último elemento, a lista fica ordenada. Cada chave é comparada com todos os elementos anteriores, sendo inserida em sua posição correta.
